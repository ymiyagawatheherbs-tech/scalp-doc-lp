## 1. 予約の流れ（全体像）

お客様がウェブサイトから予約を送信してから来店確定までの流れは以下のとおりです。

```
お客様がフォーム送信
    ↓
① 受付確認メール（自動送信）＋ LINE通知「📋 新規予約申し込み」（スタッフへ）
    ↓
② スタッフが管理ページで内容確認・日程調整（電話またはメール）
    ↓
③ 管理ページでステータスを「確定」に変更
    ↓
④ 予約確定メール（自動送信）→ お客様へ
   ＋ LINE通知「✅ 予約確定」（スタッフへ）
    ↓
⑤ 来店当日

※ キャンセルの場合：ステータスを「キャンセル」に変更
   → LINE通知「❌ 予約キャンセル」（スタッフへ）
```

## 3. 管理ページの操作手順

### 3-1. 管理ページにアクセスする

ブラウザで以下のURLにアクセスし、スタッフアカウントでログインします。

```
https://scalp-labo.jp/admin
```

ログインには、スタッフ用のメールアドレスとパスワードが必要です。初回ログイン情報は管理者（ゆみこ）にご確認ください。

### 3-2. 予約一覧を確認する

管理ページにログイン後、予約一覧が表示されます。各予約には以下の情報が表示されます。

| 表示項目 | 説明 |
|---------|------|
| 受付日時 | 予約フォームが送信された日時 |
| お名前 | 予約者のお名前 |
| 希望日時 | 来店希望日・時間帯 |
| コース | 選択されたコース名 |
| 電話番号 | 連絡先 |
| メール | 未記入の場合は空欄 |
| ステータス | 受付中 / 確定 / キャンセル |

新規予約はステータスが「**受付中（pending）**」になっています。

---

## 4. ステータスの種類と操作

予約には3種類のステータスがあります。

| ステータス | 意味 | 操作タイミング | LINE通知 |
|-----------|------|--------------|---------|
| **受付中** | お客様から予約が届いた初期状態 | 自動設定 | 📋 新規予約申し込み |
| **確定** | 日時・コースを確認し来店確定 | スタッフが手動で変更 | ✅ 予約確定 |
| **キャンセル** | 来店キャンセルになった場合 | スタッフが手動で変更 | ❌ 予約キャンセル |

> **重要：** ステータスを「**確定**」に変更した瞬間に、お客様のメールアドレスへ**予約確定メールが自動送信**され、スタッフのLINEにも確定通知が届きます。

## 6. 受付確認メールの内容（参考）

お客様が予約フォームを送信した直後に**自動送信**される確認メールです。スタッフの操作は不要です。

### 件名
```
【THE HERBS SCALP LABO】ご予約を受け付けました — ○○ 様
```

### 主な内容
- 受付した予約内容（店舗・コース・希望日時・お名前）
- 「担当者よりメールまたはお電話にてご連絡いたします」
- キャンセルポリシー（前日までにご連絡）
- 当日予約の電話番号（070-2642-7366）
- 店舗アクセス情報（営業時間：10:00〜20:00）
- LINEお友だち追加リンク

> **注意：** 受付確認メールはあくまで「受け付けた」という通知です。この時点では予約は確定していません。スタッフが確認・調整後に「確定」ステータスに変更することで、正式な予約確定となります。

## 9. キャンセル対応

お客様からキャンセルの連絡があった場合は、以下の手順で対応してください。

1. 管理ページにアクセス（https://scalp-labo.jp/admin）
2. 該当の予約を検索
3. ステータスを「**キャンセル**」に変更
4. 変更後、スタッフのLINEに「❌ 予約キャンセル」通知が届きます
5. 必要に応じてお客様へ電話またはメールでご連絡ください

> **注意：** キャンセル時はお客様への自動メールは送信されません。

## 10. よくある質問

**Q. メールアドレスを入力していないお客様には確定メールが届きますか？**  
A. 届きません。メールアドレスが未記入の場合は、電話にてご連絡ください。

**Q. 予約確定メールを誤って送ってしまった場合は？**  
A. 現在のシステムでは送信取り消しはできません。お客様に電話またはLINEにてご連絡ください。

**Q. 管理ページのパスワードを忘れた場合は？**  
A. 管理者（ゆみこ）にご連絡ください。パスワードの再設定が可能です。

**Q. LINEに通知が届かない場合は？**  
A. 「THE HERBS神戸阪急店」LINEアカウントが公式アカウントを友だち追加しているか確認してください。また、LINEアプリの通知設定もご確認ください。

**Q. 植物美容サロン（灘区）の予約はどこで管理しますか？**  
A. 植物美容サロンはSquare予約システムを使用しています。Squareの管理画面をご確認ください。
